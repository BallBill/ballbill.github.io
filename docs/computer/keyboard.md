### 基礎操作

| 開機 | 短按Home鍵1秒（可組合A/B/X/Y鍵打開對應連接模式） |
| --- | --- |
| 關機 | 長按Home鍵5秒關機 |
| 待機 | 手柄處於連接狀態下10分鐘無操作，則自動關機 |
| 低電提示 | 手柄電量低15%，Home燈紅色慢閃提示 【手柄低電量狀態同時關閉震動】 |
| 底座充電 | 底座連接電腦，手柄放置底座上即可充電，手柄拿起後默認關機前的模式 【充滿後手柄和底座滅燈狀態】 |
| 手柄充電 | 手柄充電可通過數據線連接手柄進行充電，關機狀態下充電時【燈帶呼吸】指示當前電量 0-20%【紅色】20%-40%【橙色】40%-74%【黃色】75%-95%【綠色】95%以上【充滿滅燈】 |
| 查看當前模式 | 單獨按住M鍵可以查看手柄當前模式 |

### 連接狀態

| 狀態 | 通道燈 | 說明 |
| --- | --- | --- |
| 連接成功 | 常亮 | 設備已經連接成功 |
| 配對狀態 | 快速閃爍 | 處於配對狀態時可以被新設備搜索、連接 |
| 回連狀態 | 慢速閃爍 | 處於回連狀態時，可以自動連接上一個連接成功的設備，同時長按【LOGO+截圖鍵】可強制切換為配對狀態 |
